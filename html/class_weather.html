Detailed Description

Weather This class works on behalf of the client class; it should have a private member variable of Vector<Metadata>, which would give access to the data (stored in the vector, required for processing). Also, the processing required should be done in this class – not in the client class. That is, the vector should not be declared in the main function, nor should the processing be performed in main.cpp. The client takes care of input/output.
